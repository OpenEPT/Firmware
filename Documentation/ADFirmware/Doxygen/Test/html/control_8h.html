Detailed Description

Control service is in charge to receive and process control messages. This service communicates with others services defined within this firmware. Communications is mostly related to the configuration of corresponding service or obtaining status messages from certain service defined within control message content. All control service interface functions, as well as public defines and data types,are defined within this header file.
